# Icons / 圖示

?> 相關 css：scss/components/`_icon.scss`

- hyUI 提供一套可快速套用的 icon 圖示，本圖示是用`svg`呈現，非網路字型技術。
- icon 白色在檔名後加`_w`，黑色 icon 則在檔名後加`_dk`，例如：`i_search_w`為白色 icon，`i_search_dk`為黑色 icon

## Icon 黑色

Icon 順序依英文字母排序
