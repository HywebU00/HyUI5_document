下載的 icon 皆由 html/css 撰寫，`fileType`為 icon 的 class,`B`為指定的顏色，中間的內容文字可以顯示該檔案的附檔名。

```html
<span class="fileType B">doc</span>
```

目前有預設以下顏色

- `blue` 藍色
- `green` 綠色
- `brown` 棕色
- `purple` 紫色
- `orange` 橘色
- `gray` 灰色
- `cyan` 青色
- `cerulean` 蔚藍色
- `red` 紅色
